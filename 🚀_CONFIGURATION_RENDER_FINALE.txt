## 🎯 ÉTAPES DE DÉPLOIEMENT

### **Étape 1 : Préparer le projet**
```bash
# Activer l'environnement virtuel
venv\Scripts\activate

# Exécuter le script de préparation
python prepare_render_deployment.py
```

### **Étape 2 : Pousser sur GitHub**
```bash
git add .
git commit -m "Configuration finale pour Render"
git push origin main
```

### **Étape 3 : Déployer sur Render**
1. Allez sur [render.com](https://render.com)
2. Créez un compte avec GitHub
3. Cliquez "New +" → "Web Service"
4. Connectez votre repository : `omaarjami-ds/midway-cafe-website`
5. Configurez :
   - **Name** : `midway-cafe`
   - **Environment** : `Python 3`
   - **Build Command** : `pip install -r requirements.txt`
   - **Start Command** : `gunicorn midway_project.wsgi:application`

### **Étape 4 : Variables d'environnement**
Dans Render Dashboard → Variables, ajoutez :
```
DEBUG=False
SECRET_KEY=NUrwUxyrzKzjN7BhpgQE7f9zb0PMZW2-Va2HNsrKcHFSVLVu1_e8iYKV1S1CfQKmPyw
ALLOWED_HOSTS=midway-cafe.onrender.com,*.onrender.com
```

### **Étape 5 : Déployer**
Cliquez "Create Web Service" et attendez 3-5 minutes.
